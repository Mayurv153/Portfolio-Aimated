# ✨ Modern 3D Interactive Portfolio

A highly interactive, performant, and visually stunning 3D personal portfolio built with modern web technologies. This project showcases advanced WebGL rendering, physics integrations, and smooth scroll animations to create an immersive user experience.

![Preview](https://github.com/user-attachments/assets/3c4557e7-6392-4928-b8a9-7b2476ef4edd)

## 🚀 Features

- **Immersive 3D Experience**: Built with `@react-three/fiber` and `@react-three/drei` for beautiful WebGL rendering.
- **Physics Integration**: Real-time physics for interactive elements via `@react-three/cannon` and `@react-three/rapier`.
- **Advanced Animations**: Smooth page transitions and scroll animations powered by `GSAP` and `Lenis` smooth scroll.
- **Performance Optimized**: Lightning-fast development and optimized production builds with `Vite`.
- **Modern Architecture**: Strongly typed with `TypeScript` and built on `React 18`.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **3D & WebGL**: [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber), [@react-three/drei](https://github.com/pmndrs/drei)
- **Physics**: [@react-three/rapier](https://github.com/pmndrs/react-three-rapier), [@react-three/cannon](https://github.com/pmndrs/use-cannon)
- **Animations**: [GSAP](https://gsap.com/), [Lenis](https://studiofreight.github.io/lenis/)

## ⚙️ Quick Start

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mayurv153/Portfolio-Aimated.git
   cd Portfolio-Aimated
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

> **Note**: This application uses trial plugins for GSAP. If you intend to use this project for production or hosting, you will need a proper GSAP Club license.

## 📜 Available Scripts

- `npm run dev` - Starts the development server with Vite.
- `npm run build` - Compiles TypeScript and builds the app for production.
- `npm run preview` - Previews the production build locally.
- `npm run lint` - Runs ESLint to check for code issues.
